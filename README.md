- Project highlights
- Contact information
- Clean and modern UI

## 🚀 Getting Started

1. Clone this repository
2. Edit the HTML files to add your content
3. Customize the CSS for your preferred styling
4. Push changes to the `gh-pages` branch or `main` branch (if configured)
5. Visit `https://<your-username>.github.io` to view your site

## 📝 Customization

To customize this site:
- Update content in HTML files
- Modify colors and fonts in `css/style.css`
- Add interactivity with `js/script.js`
